# Phaser 3 + TypeScript + Vite.js Template




```bash
git clone https://github.com/yutao721/phaser3-vite-template my-folder-name
cd my-folder-name

npm install
```

开发环境:

```
npm run start
```

打包:

```
npm run build
```



## 目录结构

```
    .
```
